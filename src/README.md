# Instruções de utilização

## Instalação do Site

Em sua primeira versão, o site foi construído em HTML/CSS/JS, podendo ser utilizado tanto em servidores quanto em sites de hospedagem estáticos (exemplo: Github Pages). Após retomada do projeto, a reestruturação do SISCAMOVEL está sendo feita utilizando as tecnologias ASP.NET CORE 7 (back-end) e Angular 15 (front-end), tendo a necessidade de utilizar serviços em nuvem (Cloud Computing) para utilização do projeto.
